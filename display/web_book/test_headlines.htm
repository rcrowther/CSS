Headlines

Headlines 1--5 are provided (several alternative sets of heading style can be uncommented, see the CSS file). All headings can handle line-wrapping:

(h1) Chapter

Following text...

(h2) Section

Following text...

(h3) SubSection

Following text...

(h4) SubSubSection

Following text...

(h5) Paragraph

Following text...

(h6) SubParagraph

Following text...
