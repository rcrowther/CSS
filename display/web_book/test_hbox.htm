HBox

Robert Crowther July 2016

The layout tools on this page position page elements horizontally. HPos and HBox break to vertical layout on a small screen. The BCrumb theme wraps.

The tools are not designed for HTML lists. Lists carry generic formatting, and add superfluous HTML elements. Use divs/spans/images etc.

HPos

While the hbox described below is good for layout like left/right positioning and variable-sized contents, if you need an hbox for buttons or oother objects, use hpos. It's simple to layout and expands full-width on a small-screen. The disadvantage, as will clearly show with the overfilled examples below, is that it wraps badly. That's why it's called HPos. It's mainly for, say, aligning a free button centrally on a page.

HBox Basics

Webbook has another concept for horizontal box layout. This is more complexity. The advantage is it can create broken positioning e.g. left/right placement. Also, it will not wrap---it simply over-expands. And it is better for the dropdown menus classes described further down.

These classes can not be used on lists. Lists come with awkward preset spacings. Lists add too much extra HTML structure, and their preset spacing is too much to kill.

Three base boxes are provided, with alignment in different directions. Contents can be near-anything, classed as 'hbox-item'. Box-items take on the size of their contents. All are responsive. They break on paper-web width.

Drop Menus

A drop menu can also be built. This needs the HTML form defined below, and the included snippet of JS.

For narrow/vertical display the usual size for menu buttons is button-sl. For narrow/vertical consistency, all buttons should be the same size. But in the horizontal direction, the horizontal buttons probably need more vertical size. hbox-button-l has the same wide-display width as button-sl and button-nl, so can be used for root menu items. Drop menu items will use button-sl or button-nl. If you want to construct your own breaking button for another horizontal width, keep it the same width as the drop menu sizes, or text alignment will break,
