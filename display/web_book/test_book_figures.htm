Illustrations

Illustrations are split into spanblacks (necessary), and figures (unnecessary). Both elements surround transformed non-phrasal items. Both HTML elements behave much the same.

Illustrations can contain images, such as the display below, with bottom caption,

and be framed. This figure has a top caption, which requires a class "ilsn-caption-top",

...or ruled,

Book illustrations are nearly always default width, but 'wide', 'narrow', 'x-narrow'. and 'xx-narrow' figure sizes are available,

Spanblocks

Spanblocks are illustrations, like figures, but essential to text. They can contain any illustrative content such as images, objects, tables...

Even quotes, in which case spanblocks are like blockquotes, but essential,

Like figures, spanblocks can set narrow, the contents framed/ruled, cited, etc.
