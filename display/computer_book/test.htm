Code blocks and illustrations

For direct illustration, use a pre--code block. These are 'pre' to allow free formatting: 

def fileForeach() =
	    for (file <- filesHere)
		       yield file

Code formatting may be used for errors, script results, also: 

.../Shapes.scala:6: error: error overriding method

A complete, independent block of code (correct or incorrect) should be a figure (here with the 'web' theme):.

class ColoredPoint(x: Int, y: Int, val color: Color.Value)
	    extends Point(x, y) {

	    override def hashCode = 41 * super.hashCode + color.hashCode
	    }
