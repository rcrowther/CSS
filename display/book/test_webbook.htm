CSS for articles and books

Robert Crowther July 2016

When webbook.css is placed after book.css, it adds some specific web-orientated features.

Text Style

Text is switcheed to a generic serif font. It is slightly greyed.

Responisve design

Using media queries, this page is noww responsive, snapping to device widths.

Figured images

Figured images in large block sizes are locked to rounded pixel sizes. At worst they will be 1px wrong size. But rounding enables generation of image sources to the size.

xx-narrow reduces to dimensions 0x0 on small screens/thin blocks, so is expanded to x-narrow.

Don't forget to close the 'article' or 'section'.

TOC themes

This markup is set in a new 'section'.

Book.css offers a display-only table of contents (TOC). Webbook.css extends this to offer themes that will work with embedded links/anchor elements.
